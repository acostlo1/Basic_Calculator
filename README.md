# Basic Calculator (Java)

## 📌 Description
A console-based calculator that performs basic arithmetic operations using user input. This project demonstrates core Java fundamentals such as control flow, switch statements, and input validation.

## 🚀 Features
- Supports addition, subtraction, multiplication, division, and exponentiation
- Handles division by zero safely
- Validates operator input
- Uses switch expressions for clean decision-making

## 🛠️ Built With
- Java
- Scanner (for user input)
- Math.pow() (for exponent operations)

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ, Eclipse, or any Java IDE
3. Run `Main.java`
4. Follow the prompts in the console

## 📚 What I Learned
- How to use switch statements for multiple conditions
- Input handling with Scanner
- Validating user input and preventing errors
- Working with operators and basic math logic
- Using boolean flags to control program flow
